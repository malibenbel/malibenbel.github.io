Boucheron Strategic Analysis

This project delivers a full-scale strategic data analysis of the luxury brand Boucheron, leveraging multiple data sources to uncover insights into product evolution, pricing trends, and market positioning.

The project integrates over 5 million data rows in Google BigQuery, enriched through API calls and web scraping of Boucheron’s official site. Data was cleaned and processed using Python and SQL, then visualized in an interactive Power BI dashboard designed for strategic decision-making.

Key analyses include catalog shifts, pricing volatility across markets, and potential impacts of exchange rate fluctuations on perceived brand value.

Power Plant Analysis (MATLAB)

Facebook Social Network Analysis

This project explores the structure of a Facebook friendship network using data collected via the Facebook Graph API.

Key analyses include centrality measures, cluster detection, and community visualization through Gephi. Python was used to clean and preprocess the data, while the network was visualized through modularity and betweenness centrality.

Financial Analysis of Industry Giants

This project benchmarks two global companies—Walmart and Novartis—through detailed financial ratio analysis, vertical/horizontal statements analysis, and industry comparison.

Data was processed in Excel and further visualized using Python to track profitability, liquidity, and capital structure trends over time.

NLP Reddit Comments

This project implements an NLP pipeline for Reddit comment analysis using Python.

It includes advanced text cleaning, sentiment analysis with VADER, topic modeling via LDA, and word cloud visualization. Results reveal community sentiment trends and discussion patterns within specific Reddit threads.
